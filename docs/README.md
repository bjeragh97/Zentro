# Zentro Documentation

Purpose: Lightweight, practical docs for a solo developer. Keep this current, decision-focused, and easy to scan.

## How to use this folder
- Start with `project-status.md` for the current milestone and risks
- Use `setup-local.md` to get running locally
- Use `runbooks/` for quick fix-it workflows
- Record decisions in `decisions.md`
